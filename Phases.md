# ZAD PROJECT — DEVELOPMENT PHASES

## OVERVIEW

The project is built in **strict sequential phases**.  
Each phase depends on the previous one and must be fully completed before moving forward.

Agents MUST follow this order and MUST NOT skip or mix phases.
